// methodology

How It Works

01
📥

Data Loading

Video is uploaded and validated. Frame count, FPS, and duration are extracted using OpenCV.

02
🖼️

Frame Extraction

16 evenly-spaced frames are sampled from the video, resized to 128×128, and normalized to [0,1].

03
🔬

Preprocessing

Noise reduction, pixel normalization, and motion stabilization prepare frames for model inference.

04
🧠

GRU Analysis

Gated Recurrent Units analyze the frame sequence for temporal anomalies and pixel-level inconsistencies.

05
📊

Classification

The model outputs a confidence score. Videos above the threshold are flagged as deepfake.
